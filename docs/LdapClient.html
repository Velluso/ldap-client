Class: LdapClient

LdapClient(config)

Class representing a client for interacting with an LDAP server. This class provides methods to connect to the LDAP server, bind users, retrieve user information, and check authorization.

Constructor

new LdapClient(config)

Creates an instance of the LDAP client.
Parameters:
Name Type Description
config LdapConfig Configuration for the LDAP client. Users can specify the following: - `url`: The LDAP server URL. - `userBase`: The base Domain Components (DC) for user searches. - `authorizedGroups`: A list of groups that are allowed access.
