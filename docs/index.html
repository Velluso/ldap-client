LDAP-Client Documentation

This library provides a simple interface for interacting with LDAP servers, allowing operations such as authentication, retrieval of user information, and group management.

Purpose of the Library

The LDAP library enables developers to easily connect and interact with LDAP servers to manage user-related information. The main features include:

  • Connection to the LDAP server - Establish a connection to the server.
  • User Authentication - Verify user credentials using the bind method.
  • Information Retrieval - Obtain details such as full names, groups, and email addresses of users.
  • Authorization Check - Verify if a user belongs to authorized groups for modular and controlled access to a system.

Prerequisites

Before using this library, ensure that you have Node.js installed. You can download it from nodejs.org.

Installation

To install the LDAP-Client library, you can use npm (Node Package Manager). Run the following command in your terminal:

npm install ldap-client

Environment Variables

It is crucial to properly configure the environment variables before using this library. Make sure to define the following variables in your environment:

const config = {
    url: process.env.LDAP_URL,                  // URL of the LDAP server
    userBase: process.env.LDAP_USER_BASE,       // Base Domain Components (DC) for user searches (e.g., `dc=example,dc=com`)
    authorizedGroups: process.env.LDAP_AUTHORIZED_GROUPS // Comma-separated list of authorized groups
};

Usage

To get started with the library, import the LDAP client and create an instance of the LdapClient class with the appropriate configuration:

const LdapClient = require('ldap-client');

const config = {
    url: process.env.LDAP_URL,
    userBase: process.env.LDAP_USER_BASE,
    authorizedGroups: process.env.LDAP_AUTHORIZED_GROUPS
};

const client = new LdapClient(config);

Connecting to the LDAP Server

Next, you can connect to the server and authenticate users:

async function authenticateUser(username, password) {
    try {
        await client.connect();               // Connect to the LDAP server
        await client.bind(username, password); // Bind (authenticate) the user
        console.log('User authenticated successfully!');
    } catch (error) {
        console.error('Authentication failed:', error);
    }
}

// Call the function with a username and password
authenticateUser('domain\\username', 'your_password');

Retrieving User Information

After authentication, you can retrieve user details such as groups and email:

const groups = await client.getUserGroups(username);
const email = await client.getUserEmail(username);
console.log('User Groups:', groups);
console.log('User Email:', email);

Authorization Check

Check if the authenticated user belongs to authorized groups:

if (client.isAuthorized(groups)) {
    console.log('User is authorized.');
} else {
    console.log('User is not authorized.');
}

Disconnecting from the LDAP Server

Finally, disconnect from the LDAP server:

await client.unbind();
